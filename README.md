# 🔐 Random Password Generator

## 📌 Project Overview

The Random Password Generator is a Python-based application developed as part of the DecodeLabs Internship Program. This project generates secure and random passwords based on the length specified by the user.

The application uses Python's built-in `random` and `string` modules to create strong passwords containing uppercase letters, lowercase letters, numbers, and special characters.

---

## 🎯 Objectives

- Learn how to use Python modules.
- Understand string manipulation techniques.
- Generate secure passwords programmatically.
- Improve problem-solving and logical thinking skills.

---

## ✨ Features

- User-defined password length.
- Random password generation.
- Includes:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Special characters (!, @, #, $, %, etc.)
- Password strength indicator.
- Input validation for incorrect values.
- Easy-to-use command-line interface.

## ▶️ How to Run

1. Install Python 3.
2. Download or clone the repository.
3. Open terminal or command prompt.
4. Navigate to the project folder.
5. Run the following command:

```bash
python password_generator.py
```
